Privacy Policy for D&D Log Keeper

Effective Date: June 29, 2025

1. Overview

D&D Log Keeper is a session-based logging API hosted on Replit. This API only records gameplay-related logs and does not collect personal or sensitive user data.

2. Data Collection

We only collect:

  • Session IDs provided by the player
  • Narrative log content submitted by the player
  • Optional NPC names, tags, and relationship changes provided by the player

We do not collect usernames, emails, IP addresses, or personally identifiable information.

3. Data Storage

All logs are stored locally on Replit's servers. Logs are not shared or transmitted to third parties and can be deleted by the player at any time using the provided reset endpoints.

4. Third-Party Services

This service is hosted via Replit. Please review Replit's Privacy Policy for their server-level data practices.

5. User Control

Players can:

  • Create logs
  • Retrieve logs
  • Delete all logs or specific session logs at any time

6. Security

This API is designed for tabletop gaming and sandbox simulation use. Users should not submit sensitive or private information to this service.
